Modeling Module¶

Modeling.CV_Evaluate(dataset, label, model, parameter={'default': 'default'}, outer_folds=5, inner_folds=5, metrics_include={'Accuracy': 'accuracy', 'F1 Score': 'f1', 'Precision': 'precision', 'ROC AUC': 'roc_auc', 'Specificity': make_scorer(recall_score, pos_label=0)}, metrics_optim='Accuracy', print_process=True, print_result=True)¶

Evaluate the given model’s performance on the given dataset based on the given set of parameters through Nested K Folds Cross Validation. The resulting performance gives a theoretically best representative evaluation of the model.

Parameters

  • dataset (Pandas DataFrame) – Input dataset to train and evaluate the model.

  • label (string) – Label name of the given dataset.

  • model (sklearn model object) – Model to train and evaluate.

  • parameter (dictionary) – A set of hyperparameters for the model. If default, call the get_default_param() function to get the pre selected set of parameters for the model.

  • outer_folds (integer) – number of folds for the outer CV. Outer CV is for performance evaluation. Default is 5 folds.

  • inner_folds (integer) – number of folds for the inner CV. Inner CV is for hyperparameter tunning. Default is 5 folds.

  • metrics_include (dictionary) – Mertics to include during the training and evaluation. Default includes Accuracy, ROC AUC, Precision, Specificy, and F1 Score.

  • metrics_optim (str) – Metric used for optimizing the model. Default is only using Accuracy.

  • print_process (boolean) – Print the training process or not. Default is True.

  • print_result (boolean) – Print the CV evaluation result or not. Default is True

Returns

Resulting evaluation

Return type

dictionary

Modeling.CV_best_parameter(dataset, label, model, num_folds=5, metrics='accuracy', parameter={'default': 'default'})¶

Produce the best set of hyperparameters through Gridsearch with Cross Validation, given the model and the dataset.

Parameters

  • dataset (Pandas DataFrame) – Input dataset.

  • label (string) – Label of the given dataset.

  • model (sklearn model object) – Model to train and search for the best set of hyperparameter.

  • num_folds (integer) – Number of folds for CV, default is 5.

  • metrics (string) – Metric used for model optimization, default is Accuracy

  • parameter (dictionary) – The set of parameters to search from, default set is provided by the get_default_param() function.

Returns

The best set of hyperparameters.

Return type

dictionary

Modeling.evaluate_model(dataset, label, model, metrics={'Accuracy': 'accuracy', 'F1 Score': 'f1', 'Precision': 'precision', 'ROC AUC': 'roc_auc', 'Specificity': make_scorer(recall_score, pos_label=0)}, num_folds=None)¶

Evaluate the given model on a given dataset. When num_folds is None, Cross Validation is not used, and the whole input dataset is used for evaluation.

Parameters

  • dataset (Pandas DataFrame) – Dataset to evaluate the model.

  • label (string) – Label.

  • model (sklearn model object) – Model to evaluate on the dataset.

  • metrics (dictionary) – Metrics to include in the evaluation. Default includes Accuracy, ROC AUC, Precision, Specificy, and F1 Score.

  • num_folds (integer) – Number of folds for the Cross Validation. Default is None, which uses the whole dataset for evaluation.

Returns

Evaluation results of the model
