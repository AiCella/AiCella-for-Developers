Visualization Module¶

The visualization.py includes functions for creating both interactive and static visualizations using a combination of Matplotlib, Seaborn, Bokeh, and Plotly.

It has to be used with the Modeling.py file in the same directory.

Visualization.feature_importance_permutation(model, dataset, label, n_repeats=7, bokeh_figsize_x=700, bokeh_figsize_y=500, bar_width=1, color='skyblue', alpha=0.7, label_rotate=1.2)¶

Creates a bar plot to visualize the feature importance through Feature Permutation, given a model and a dataset. Feature Permutation provides a more precise, unbiased estimation of feature importance, at the cost of higher computational resources.

Parameters

  • model (sklearn model object) – Input model, available for all sklearn model objects.

  • dataset (Pandas DataFrame) – Input dataset to evaluate on, must have the same feature set with the same column order as the dataset that model was trained on.

  • label (string) – Target label.

  • n_repeats (integer) – Number of times to repeat the permutation. Default is 7.

  • bokeh_figsize_x (float or integer) – Figure size on the X axis, unit is in pixels instead of inches. Default is set by the global bokeh x size value.

  • bokeh_figsize_y (float or integer) – Figure size on the Y axis, unit is in pixels instead of inches. Default is set by the global bokeh x size value.

  • bar_width (float or integer) – Width of the bars in chart. Default is 1.0.

  • color (string) – Color of the bar and their outlines. Default is skyblue.

  • alpha (float or integer) – Color alpha for the bar’s filling color. Default is 0.7.

  • label_rotate (float or integer.) – Rotation of x axis labels. Default is 1.2.

Returns

Resulting bar chart to visualizae input model’s feature importance on the given dataset.

Visualization.tree_prediction_paths(dataset, label, model, instance, scale=2, orientation='TD', show_distribution=True, simplified=False, class_names=['Negative', 'Positive'])¶

Creates a diagram to visualize Decision Tree prediction path for a specific instance.

Parameters

  • dataset (Pandas DataFrame) – Dataset the model was trained on.

  • label (string) – Target label.

  • model (sklearn model object) – Input model, must be a Decision Tree model.

  • instance (Series or DataFrame) – Input instance, must be a single entry with the same format as the training dataset.

  • scale (float or integer) – Scale of the output chart. Default is 2.0.

  • orientation (string) – Orientation of the output chart. Choose one between “TD” (Vertical) and “LR” (Horizontal). Default is Vertical.

  • show_distribution (boolean) – Display feature distribution or not. Default is True.

  • simplified – Display a simplified tree prediction path, only includes the path that instance took to reach the prediction. Default is False, which means includes all paths of the tree structure and highlighs the path taken by the instance.

  • class_names (list of strings) – Class names of the label class. Default is for binary classification, [“Negative”, “Positive”].

Returns

Resulting tree structure diagram object.

  • class_names (list of strings) – Class names of the label class. Default is for binary classification, [“Negative”, “Positive”].

Returns

Resulting tree structure diagram object.

Visualization.xgb_feature_importance(model, bokeh_figsize_x=700, bokeh_figsize_y=500, bar_width=1, color='skyblue', alpha=0.7, label_rotate=1.2)¶

Creates a bar plot to visualize the feature importance for a given XGBoost model.

Parameters

  • model (sklearn model object) – Input model, must be a XGBoost model.

  • bokeh_figsize_x (float or integer) – Figure size on the X axis, unit is in pixels instead of inches. Default is set by the global bokeh x size value.

  • bokeh_figsize_y (float or integer) – Figure size on the Y axis, unit is in pixels instead of inches. Default is set by the global bokeh x size value.

  • bar_width (float or integer) – Width of the bars in chart. Default is 1.0.

  • color (string) – Color of the bar and their outlines. Default is skyblue.

  • alpha (float or integer) – Color alpha for the bar’s filling color. Default is 0.7.

  • label_rotate (float or integer.) – Rotation of x axis labels. Default is 1.2.

Returns

Resulting bar plot to visualize the feature importance.
