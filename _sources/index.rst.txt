Welcome to AiCella for Developers's Documentation!
==================================================
This documentation is intended for advanced AiCella customers who wants to create their own custom Machine Learning pipelines. It includes two modules: the Modeling module which contains all functions related to ML modeling and analytics, and the Visualization module which contains all visualization related functions. Please be aware that the functions are still under active development, and new features will be added rapidly. 

Required Packages
--------------------
In order to run the provided Python Modules properly, the following Python packages are required:

**Modeling Module:**

.. code-block::

   ->	pandas		1.3.4
   ->	numpy		1.21.6
   ->	sklearn		1.0.1
   ->	lightgbm	4.2.0
   ->	joblib		1.1.0
   ->	xgboost		1.6.1

**Visualization Module:**

.. code-block::

   ->	pandas		1.3.4
   ->	numpy		1.21.6
   ->	seaborn		0.12.2
   ->	bokeh		2.4.1
   ->	sklearn		1.0.1
   ->	matplotlib	3.5.3
   ->	numpy		1.21.6
   ->	plotly		4.12.3
   ->	dtreeviz	2.0.0
